# A-simple-desktop-note
A lightweight Python desktop note-taking program.  
This program is my first open source project!  

<br>

### Keyboard shortcut
<p> Ctrl + S: save your note </p>
<p> Ctrl + R: move right bottom corner of the screen </p>
