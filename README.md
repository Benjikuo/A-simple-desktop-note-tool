# A-simple-desktop-note
<p> A lightweight Python desktop note-taking program. </p>
<p> This program is my first open source project!</p>

<br>

### Keyboard shortcut
<p> Ctrl + S: save your note </p>
<p> Ctrl + R: move right bottom corner of the screen </p>
